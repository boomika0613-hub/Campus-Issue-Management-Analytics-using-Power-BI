# Campus Issue Analysis Dashboard using Power BI
## Project Objective

The main objective of this project is to analyze campus-related issues using Power BI and identify patterns, trends, and problem areas.
The project converts raw issue data into meaningful visual insights that help authorities understand issue frequency, types, and resolution status.
This dashboard supports data-driven decision-making to improve campus facilities and services.

## Dataset Used

-<a href="https://github.com/boomika0613-hub/Campus-Issue-Management-Analytics-using-Power-BI/blob/main/Campus_Issue_Data_1000_Rows.xlsx">Dataset</a>

The dataset used in this project is a Campus Issue Dataset stored in Excel format.
It contains records related to different types of campus issues such as maintenance, infrastructure, cleanliness, and other complaints along with their status and location.
The dataset is used as the primary source for analysis and visualization in Power BI.


## Key Questions / KPIs

Total number of campus issues reported

Issue distribution by category

Issue status (Open, In Progress, Resolved)

Most frequently reported issue types

Location-wise issue analysis

Resolution rate of issues

Monthly or time-based issue trends

High-priority problem areas

## Project Process

Imported campus issue data from Excel into Power BI.

Cleaned and prepared the data using Power Query Editor.

Created necessary relationships and DAX measures.

Defined KPIs based on campus issue analysis.

Designed interactive dashboards using charts and slicers.

Analyzed insights and finalized the Power BI report.

## Project Insights

The dashboard highlights the total number of campus issues and their current status.

Certain issue categories occur more frequently, indicating priority areas.

Location-wise analysis identifies campus zones with higher issue counts.

Resolution tracking helps monitor operational efficiency.

Interactive filters allow detailed exploration of specific issues.

## Final Conclusion

This Power BI project effectively transforms campus issue data into actionable insights.
The dashboard helps stakeholders quickly identify problem areas and track issue resolution status.
Overall, the project demonstrates strong skills in data cleaning, KPI creation, and interactive dashboard development using Power BI.
